# 心语 · 心理健康访谈系统

一个使用 DeepSeek API 的心理健康筛查访谈应用，包含黑色风格响应式聊天界面。它用于支持性信息收集，不替代专业诊断或治疗。每次生成访谈回复后，独立的终止意图判断智能体会判断访谈是否结束，并在结束时锁定输入区域。

## 启动

```bash
python -m pip install -r requirements.txt
export DEEPSEEK_API_KEY="你的 API Key"
python src/app.py
```

浏览器访问 <http://127.0.0.1:8000>。

可选环境变量：

- `LLM_MODEL`：模型名称，默认 `deepseek-v4-flash`
- `HOST`：监听地址，默认 `127.0.0.1`
- `PORT`：监听端口，默认 `8000`

## 安全说明

API Key 仅由后端读取，不会发送到浏览器。系统提示词包含危机安全回应规则，但本应用不是紧急服务；正式部署前应根据服务地区补充当地危机热线、隐私政策、数据保存策略和人工转介流程。
